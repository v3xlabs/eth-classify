## Installation

Using `npm`:

```sh
npm install eth-classify
```

or if you prefer to use the `yarn` package manager:

```sh
yarn add eth-classify
```

or if you prefer to use the `pnpm` package manager:

```sh
pnpm add eth-classify
```
